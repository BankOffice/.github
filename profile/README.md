## 프로젝트 구조
```
🌏Client 🌍
├─src
│ ├─assets (이미지, 폰트 등)
│ ├─frontDB (front 가공 DB)
│ ├─components (컴포넌트)
│ ├─pages (페이지)
│ ├─lib (기타 라이브러리)
```
